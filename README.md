# OpenCV Basics with Python

A collection of beginner-friendly OpenCV programs written in Python. This repository covers the fundamental image and video processing operations using OpenCV.

## Topics Covered

- Reading and displaying images
- Reading videos
- Saving images
- Getting image dimensions
- Converting images to grayscale
- Resizing images
- Image slicing (ROI)
- Rotating images
- Flipping images
- Drawing lines
- Drawing rectangles
- Drawing circles
- Adding text to images
- Reading video frame-by-frame
- Writing and saving videos
- Small practice projects

## Requirements

- Python 3.x
- OpenCV (`opencv-python`)

Install OpenCV using:

```bash
pip install opencv-python
```
